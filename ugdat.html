Unidade de Gestão de Dados Analíticos Tributários (UGDAT)

 
Gestor: Gerson Luiz dos Santos

Competências da UGDAT:


  • Gerir o sistema de banco de dados não analítico, definindo a arquitetura, o planejamento de implementação e de manutenção, bem como acompanhando o desempenho deste ambiente;
  • Acompanhar o desempenho do ambiente de banco de dados analíticos;
  • Definir e implementar os fluxos de uso de dados analíticos em todos os níveis organizacionais no âmbito da Administração Tributária;
  • Definir e implementar ações que visem à otimização do processo de carga de dados, bem como realizar processamentos complementares, a fim de melhorar a qualidade e desempenho das bases de dados analíticas;
  • Definir e implementar rotinas de auditoria que visem à conciliação dos dados constantes das bases analíticas com os dados originários das bases de dados transacionais;
  • Executar outras atividades que lhe forem atribuídas na sua área de atuação.
